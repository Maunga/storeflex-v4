# SEO Assets for Storeflex

This folder should contain the following images for optimal SEO:

## Required Images

### 1. Open Graph Image (`og-image.jpg`)
- **Size**: 1200 x 630 pixels (recommended by Facebook/LinkedIn)
- **Format**: JPG or PNG
- **Purpose**: Displayed when your site is shared on social media
- **Content suggestions**:
  - Include your Storeflex logo
  - Show "Dubai to Zimbabwe" shipping concept
  - Use brand colors
  - Add tagline: "Premium Products from Dubai"

### 2. Apple Touch Icon (`apple-touch-icon.png`)
- **Size**: 180 x 180 pixels
- **Format**: PNG
- **Purpose**: iOS home screen icon

### 3. Favicon (`favicon.ico`)
- **Location**: `/public/favicon.ico`
- **Sizes**: Include 16x16, 32x32, 48x48
- **Purpose**: Browser tab icon

## Recommended Additional Assets

### Twitter Card Image
- If different from OG image, create `twitter-image.jpg`
- **Size**: 1200 x 600 pixels

### Logo Variations
- `logo.png` - Full color logo (already exists)
- `logo-white.png` - White version for dark backgrounds
- `logo-dark.png` - Dark version for light backgrounds

## Image Optimization Tips

1. Compress all images using tools like TinyPNG or Squoosh
2. Use descriptive filenames (e.g., `storeflex-dubai-zimbabwe-dropshipping.jpg`)
3. Add proper alt text in code when using images
4. Keep file sizes under 200KB for OG images
